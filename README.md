## DOM Card Project

A simple and interactive DOM Card Project built using HTML, CSS, and JavaScript.
This project demonstrates how to dynamically create and manipulate card elements using DOM operations.


 ## Features

- Dynamic card creation using JavaScript
- DOM manipulation (create, update, delete elements)
- Clean and responsive UI
- Beginner-friendly project
- Interactive card layout


## Tech Stack

- HTML
- CSS
- JavaScript (DOM Manipulation)
